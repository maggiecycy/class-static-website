This project is a static website built as part of a web development course assignment.  
It demonstrates structured HTML, modular CSS styling, and interactive UI components.

本项目为 Web 开发课程中的静态网页实践作品，  
主要展示了结构化 HTML 设计、模块化 CSS 样式系统，以及基础交互组件的实现。

---

## ✨ Features | 项目特点

- Custom CSS variable color system  
  自定义 CSS 变量色彩系统
- Glass-style fixed navigation bar  
  毛玻璃效果固定导航栏
- Collapsible resume module  
  可折叠简历展示模块
- Integrated GitHub & Portfolio links  
  集成 GitHub 与个人作品集链接
- Local demo fallback (video recording)  
  提供录屏演示作为访问备用方案
- Clean spacing and layout optimization  
  注重版式间距与视觉层级优化
